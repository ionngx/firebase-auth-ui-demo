Avatar

The Avatar component allows you to display the authenticated status of users and optionally a user focused popup. It supports two modes, Default and Simple. In both modes the default behaviour is to display the icon as a placeholder when the user is not signed in. However you can change this to hide the Avatar component when the user is not signed in.

A common use case for the Avatar component is to provide the user with a simple way to sign in/out and see their status from a heaader or navigation bar as this demo application does.
