Sign Up

The Sign U[] component provides a single user interface with inputs to create a Firebase User with email and password.

Try It Out

An instance of the Sign Up component is declared directly below this paragraph. The attributes you can use to control it are detailed below. Use the controls below to test the effect of different settings.

Usage

This page declares the Sign Up component in the template like this

    <ionngx-firebase-auth-ui-sign-up></ionngx-firebase-auth-ui-sign-up>
