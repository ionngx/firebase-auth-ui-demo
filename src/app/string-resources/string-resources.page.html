String Resources

As part of configuring this library you can provide a set of string resources that will be used as alternatives for text displayed in components. To do this you simply set the stringRources property of the configuration object to an instance of the StringResources interface. The StringResrouces type comprises the following properties and default values.

Property Alternative/Translation For
cancel Cancel
email Email
password Password
profile Profile
signIn Sign In
signInFailureMessage An unexpected error occured, unable to sign you in.
signInMessage Sign in with your email and password or use one of the buttons to sign in with another account provider.
signInSuccessMessage You were successsfully signed in.
signOut Sign Out
signUp Sign Up
signUpFailureMessage An unexpected error occured, unable to create an account.
signUpMessage Complete and submit the form below to create an account.
signUpSuccessMessage Your account was successfully created.
